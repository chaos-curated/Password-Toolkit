# Password Toolkit

A beginner-friendly Python project containing two useful password utilities.

---

## Features

### Password Generator

- Generate secure random passwords
- Custom password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Guarantees at least one character from every category

### Password Strength Checker

- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects digits
- Detects special characters
- Detects common passwords
- Rates password strength

## How to Run

Clone the repository

```bash
git clone https://github.com/chaos-curated/Password-Toolkit.git
```

Run either program

```bash
python password_generator.py
```

or

```bash
python password_strength_checker.py
```
